# Tugas Besar PBO  
## Sistem Manajemen Perpustakaan SMPN 18 Sijunjung

Repository ini berisi source code dan database dari **Tugas Besar Pemrograman Berorientasi Objek (PBO)**  
yang dikembangkan sebagai pengganti Ujian Akhir Semester (UAS).

## Studi Kasus
Objek yang digunakan adalah **Perpustakaan SMPN 18 Sijunjung**, dengan fokus pada pengelolaan
data buku, peminjaman, pengembalian, serta perhitungan denda keterlambatan.

## Teknologi yang Digunakan
- Java (Object-Oriented Programming)
- MySQL Database
- JDBC (MySQL Connector)
- Git & GitHub

## Struktur Folder
- `src/` : Source code program Java
- `lib/` : Library MySQL Connector
- `db_manajemen_perpustakaan.sql` : File database
- `.vscode/` : Konfigurasi editor

## Fitur Utama
- Login & Register Admin
- CRUD Data Buku
- Peminjaman Buku
- Pengembalian Buku + Denda
- Laporan Data Peminjaman
